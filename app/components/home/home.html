CASCADE

CASCADE aides analysts in hunting advanced cyber adversaries, by using MITRE's ATT&CK framework.

CASCADE automates significant portions of the investigation process, and identifies ATT&CK techniques. Context is automatically gathered and graphs of activity are labelled with ATT&CK data, allowing analysts can focus efforts on making intelligent decisions, response and remediation.

BRAWL Data Set

NOTE: This instance of CASCADE only has read functionality and is loaded with data from BRAWL. The full source is hosted on Github and contains links to public BRAWL data sets.
